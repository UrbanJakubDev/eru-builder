GVE (Generátor výkazů ERU) is a tool for generating JSON files for ERU-E1 and ERU-T1 reports. The application allows easy conversion of data from Excel files to the format required by the Energy Regulatory Authority.

## Installation

Install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
